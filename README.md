# Assembly Not Included
AIO UE4SS mod for Drive Beyond Horizons

Version: 0.1.0-Alpha
Status: Work in progress; this is an early alpha build and features may change as development continues.

## Required UE4SS build
Use the latest experimental/dev build of UE4SS for this mod. The mod is designed for the current UE4SS development branch and may not work correctly with older stable releases.

Recommended download:
- Download the latest UE4SS build whose filename starts with zDev from: https://github.com/UE4SS-RE/RE-UE4SS/releases
- Prefer the newest zDev prerelease/dev build that matches your game version and platform.

## Installation
1. Install the latest experimental UE4SS dev build for Windows.
2. Place the AssemblyNotIncluded folder into your game's UE4SS Mods directory:
   - ...\DriveBeyondHorizons\Binaries\Win64\ue4ss\Mods\
3. Ensure the mod folder contains enabled.txt and Scripts/main.lua.
4. Start the game and verify UE4SS loads the mod from the Mods folder.

## Notes
- If the game crashes or the mod fails to load, confirm you are using the latest experimental UE4SS build and that your UE4SS version is compatible with the game build.
- This mod expects UE4SS to be installed and enabled before launching the game.
